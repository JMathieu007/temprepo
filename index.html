What this page demonstrates

This site is intentionally small, but it touches the main pieces of a basic front-end setup: structure, styling, and behavior.

HTML: structure

Semantic sections like <header>, <section>, and <footer> keep the page organized and easier to read‚Äîfor both humans and browsers.

CSS: styling

A mix of a global style.css file and a few page-specific rules add the dark theme, glowing cards, and fun colors you see here.

JavaScript: interaction

A small script.js file listens for the DOM to load, tweaks the main heading, and hooks up the button so it can respond when you click it.

Bootcamp checklist

The page can be used to show that a few common front-end skills have been practiced:

  • Create a multi-section layout with semantic HTML.
  • Link an external stylesheet and JavaScript file.
  • Add custom colors, typography, and simple responsive behavior.
  • Use JavaScript to react to a button click.
  • Deploy the project with GitHub Pages.
